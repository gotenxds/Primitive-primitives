Class EllipsePrimitive

EllipsePrimitive is implemented to be as minimal as possible. Both border and fill are ON by default and you will need to disable them. Both color and border color are just an array of 4 floats from 0 to 1 in the RGBA format, currently we do not support materials.

  • Overrides Primitive.constructor

    • Defined in EllipsePrimitive.ts:26

    The ellipse constructor requires an ellipse center point a semiMinor and semiMajor, these are used to calculate the ellipses position and thus mandatory.

    Parameters

    • options: object

      The ellipses options.

      • Optional border?: boolean

        Defines whether we should render the border, defaults to true.

      • Optional borderColor?: number[]
      • center: any

        The ellipses center point in 3D space.

      • Optional color?: number[]
      • Optional fill?: boolean

        Defines whether we should render the fill, defaults to true.

      • Optional granularity?: number

        The angular distance between points on the ellipse in radians, smaller values for smooter ellipses, bigger values for better performence, defaults to 0.3.

      • Optional rotation?: number

        The ellipses rotation, defaults to 0.

      • semiMajorAxis: number

        The ellipses big radius.

      • semiMinorAxis: number

        The ellipses small radius.

      • Optional show?: boolean

        Defines whether we should render the ellipse, defaults to true.
